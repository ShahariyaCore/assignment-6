This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

# 🚀 Project Name: React First Project

## 📖 Description
A modern React-based web application showcasing dynamic UI components, responsive design, and interactive features.  
This project is built to practice and demonstrate core React concepts along with Tailwind CSS styling.

---

## 🛠️ Technologies Used
- **React.js** – Component-based UI development  
- **Vite** – Fast build tool for modern frontend projects  
- **Tailwind CSS** – Utility-first CSS framework for styling  
- **React Toastify** – Elegant toast notifications  

---

## ✨ Features
1. **Interactive Cart System**  
   - Add items to cart with instant feedback and toast notifications.  
   - Button text dynamically changes from *Buy Now* → *Added*.  

2. **Responsive Layout**  
   - Fully mobile-friendly design using Tailwind’s grid and flex utilities.  
   - Smooth hover animations and transitions for better user experience.  

3. **Modern Footer & Navigation**  
   - Multi-column footer with product, company, resources, and social links.  
   - Tab-based navigation with active state highlighting and cart badge counter.
